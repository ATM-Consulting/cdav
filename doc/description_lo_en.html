Dolibarr module to synchronize calendars, project tasks and address book from CalDAV/CardDAV/ICS client tools such as Mozilla Thunderbird/Lightning or Android DAVx5. Dolibarr becomes a CalDAV/CardDAV server.

Since version 2, this module allows to sync tasks, which can be created (according to setup) in projects from docs like proposals or orders.

You obtain a great tool for driving your tech teams !
